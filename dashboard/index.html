About This Dashboard

This interactive dashboard visualizes air quality measurements from multiple sensors with timestamps. It supports CSV data files with measurements for CH4, CO, O2, CO2, and NO2 from two sensors (sensor_id 0 and 1).

Expected CSV format: timestamp,sensor_id,CH4 [% LEL],CO [ppm],O2 [%],CO2 [ppm],NO2 [ppm]
